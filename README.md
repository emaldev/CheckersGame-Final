# Checkers Game in Java

A simple Checkers (Draughts) game developed using Java and Java Swing.  
This project was created as a university Object-Oriented Programming (OOP) project to practice Java classes, GUI programming, and game logic.

## Features

- 8×8 game board
- Two-player mode
- Turn-based gameplay
- Valid move checking
- Piece capturing
- King promotion
- King movement
- Winner detection
- Selected piece highlighting
- Current player's turn display
- Graphical interface using Java Swing

## Project Structure

| Class | Description |
|-------|-------------|
| `Main` | Starts the application |
| `GameFrame` | Creates the main window |
| `GamePanel` | Draws the board, pieces, and handles mouse input |
| `Game` | Controls the game flow and turns |
| `Board` | Manages the board and game rules |
| `Tile` | Represents a single square on the board |
| `Player` | Represents each player |

## Technologies

- Java
- Java Swing
- Object-Oriented Programming (OOP)

## Game Rules

- Pieces move diagonally.
- Normal pieces move one step forward.
- Pieces capture by jumping over an opponent.
- Reaching the last row promotes a piece to King.
- Kings can move diagonally in all directions.
- The game ends when one player has no pieces or no valid moves.
